2. Implement a function to find the longest common subsequence

15 min Medium JavaScript Algorithms

Problem Description

Write a function that finds the longest common subsequence (LCS) between two strings. The LCS is the longest sequence of characters that appear left to right (but not necessarily in a contiguous block) in both strings.

Examples

Inputs: str1: "ABCDGH", str2: "AEDFHR"

Output: "ADH"

Explanation: The longest common subsequence is "ADH" which appears in both strings in that order.

Inputs: str1: "AGGTAB", str2: "GXTXAYB"

Output: "GTAB"

Explanation: The longest common subsequence is "GTAB" with length 4.
